# Manual Testing

This folder contains examples of manual testing documentation created as part of my QA portfolio.

## Content
- **test-cases/** – manual test cases for login functionality
- **bug-reports/** – bug reports written in a JIRA-like format
- **test-plan.md** – test plan describing the scope and approach for testing login functionality

## Purpose
The goal of this section is to demonstrate my approach to manual testing, including test case design and bug reporting.
